Past Projects

Savoir Care Client

This is the client frontend for an online store. It enables customers to view the catalog and place orders for items. Customers need to Sign up or Login in order to perform some actions. This frontend interacts with the e-store API to retrieve all the necessary information. This project was executed using Next.js on Node.js, Tailwind CSS for the design, and other libraries.

Savoir Care Admin

This is the administrators' frontend for an online store. It enables the administrators to manage the catalog and orders. They can add new items and update existing items. This frontend interacts with the e-store API to retrieve all the necessary information and perform the business operations. This project was executed using Next with Typescript on Node.js, Tailwind CSS for the design, and other libraries.

Savoir Care API

This is a REST API server that integrates with 2 frontend websites for an online store. The API enables it's clients to retrieve information about the catalog, to place orders, and also enables the administrators to manage the catalog and perform analysis of the online store. It also provides authentication and authorization which determines the operations a client can perform. The API was built in line with REST standards, and provides a detailed documentation of how to consume it. The documentation was also created in line with OpenAPI specifications, an industry standard. Technologies used in this project include Express.js on Node.js, Passport.js for jwt-authentication, and Mongoose for interaction with the database. MongoDB was used for cloud database storage. The server is deployed on App Engine on the Google Cloud Platform (GCP).

File Organizer

This is a command line interface (CLI) application that renames the files in a folder to something orderly. Some folders may contain files that have names that are difficult to remember for humans, names that were generated automatically by a computer. This program helps to rename the files in such a folder in an orderly manner, numerically. The user can also specify a prefix to be added to the names of the files for even further organisation. A more detailed documentation is available on the project repository. This project was built with Golang.
